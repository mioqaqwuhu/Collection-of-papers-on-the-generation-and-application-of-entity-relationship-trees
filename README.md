## 📋 项目简介

本项目聚焦自然语言处理（NLP）领域的领域特定命名实体识别（DS-NER），提供通信领域专业术语数据集及配套的改进 TF-IDF 算法实现，用于解决传统方法中术语稀疏性、隐性近义关系识别不足的问题。

核心产出包括两类标注数据集与实体关系树构建工具，支持专业术语提取、近义关系挖掘及知识图谱构建等下游任务。
## 📊 数据集详情
| 数据集类型 | 规模 | 内容构成 | 核心用途 |
 |------------|------|----------|----------|
  | 小数据集 | 100篇通信领域论文 | 随机筛选的学术文献，涵盖频谱、5G波束成形、信道编码等核心主题 | 快速验证算法有效性、小样本近义关系识别 |
   | 大数据集 | 约5000篇通信领域论文 | 按主题（类型/年份/大陆）分类的大规模语料库，含完整上下文与术语共现信息 | 实体关系树构建、大规模模型训练与评估 |
-   数据格式
    -   原始格式：PDF（已转换为无嵌入式图片的 DOCX 格式）
    -   结构化存储：SQLite 数据库（含`article`、`paragraph`、`sentence`三张核心表）
    -   输出格式：JSON（术语对、相似度评分、跳数距离等结构化数据）

## 🔧 数据预处理流程

1.  格式转换：PDF → DOCX（过滤嵌入式图片，保留纯文本）
2.  语料解析：提取标题、段落、句子，过滤参考文献部分（文本长度＜20 字符的内容已剔除）
3.  术语筛选：通过 NLTK 进行词性标注，保留名词 / 专有名词，过滤停用词
4.  结构化存储：按国家、期刊、年份分类，构建含术语共现信息的数据库

## 🎯 核心特性与适用场景

### 核心特性

-   覆盖通信领域高频专业术语（如`wavelength`、`bandwidth`、`channel`等）
-   提供术语间**三维位置特征（x - 句子级 /y - 段落级 /z - 文章级）** 与相似度评分
-   支持 "跳数距离" 量化术语近义关系强度（跳数越少，关系越强）

### 适用场景

-   领域特定命名实体识别（DS-NER）算法研究与验证
-   专业术语近义关系挖掘与词典构建
-   实体关系树 / 知识图谱自动化构建
-   改进型 TF-IDF 算法优化与对比实验
-   通信领域文本语义分析与智能检索
